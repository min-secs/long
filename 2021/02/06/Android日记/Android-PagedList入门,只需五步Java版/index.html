####不同DataSource实现类的区别
目前有三种DataSource实现方式, 上面使用的是PositionalDataSource, 想必你已经看到效果了

  • PositionalDataSource
    适用于上拉加载更多的场景, 如京东商城列表浏览, 知乎等
  • ItemKeyedDataSource
    适用于下一part内容依赖当前某个key来获取, 如获得下一part的专辑列表, 需要传入当前歌手来获取
  • PageKeyedDataSource
    适用于有明显上下页的场景, 如支持上一章,下一章跳转可以使用此种方式
