Android string.xml 插值的简单应用

在string.xml中定义

 <string name = “myname”>他今年%1$d岁了</string>

在程序中

String string = context.getResource().getString(R.string.myname);

MyTextView.setText(String.format(string,8));

打印 他今年8岁了

%d 整数

%f 浮点数

%s字符串

也可以在String.xml中%2添加多个值，依次在代码中填入相应值即可
